git init - começar repositório

git status - verificar o estado do repositorio(arquivos que estão e que não estão no repositório)

git add "nome_arquivo" - para adicionar arquivo para commit

git add . - para adicionar todos os arquivos para adicionar arquivo para commit

git commit - adicionar ao repositorio

-m - adicionar mensagem/comentario. Ex: git commit -m "commit inicial"

git remote add origin url - para adicionar repositorio github para fazer push dos commits

git push - mandar para o repositorio do github

git reflog - hitórico de versões

git reset --hard id - recuperar versão anterior

git branch nome - criar branch

git chekout nomeDaBranch - mudar de branch
